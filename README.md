VidXtract – Neon Edition

⚡ Multi-Platform Video Downloader UI (Client-Side • GitHub Pages Ready)

VidXtract is a neon-themed instant downloader interface supporting auto URL detection, thumbnail previews, smart resolution selection, fake progress animation, rewarded popup, floating video preview, and local download history — all fully front-end, no backend required.
